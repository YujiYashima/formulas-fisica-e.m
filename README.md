<h2 id="funcoes">Funções</h2>

Exemplos:

```js
const fisica = require('physics-formulas-hs')

fisica.velocidadeEscalarMedia(10, 2, "km/h") // --> 5 km/h

fisica.aceleraçaoMedia(100, 5) // --> 20 m/s^2

fisica.funçaoHorariaDaVelocidade(10, 20, 5) // --> 110 m/s
```
